Web front-end or web-slash-f or simply <web/>F is an initiative for passionate front-end-developers.

This time we are going to build enterprise web application with Angular.js. Topics will include Angular, build tools (Gulp, etc.), TDD (Mocha, Jasmine, etc.), code organization, async programming, functional JavaScript and much more.

Side note: Learning Angular.js and Building an app with Angular.js are two different things. If you wish to understand this difference, then you a perfect candidate to attend the workshop.
